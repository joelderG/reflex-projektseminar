Introduction

This Angular demo project showcases how to use the different styles and components provided by @reflex/angular-components Component Library located in /packages/shared-angular-components.

This landing page should provide simple usage instructions (also documented in readme.md) and examples for the provided components (including spec regarding the css layouting).

  1. run npm install from the workspace root.

    HINT:to cleanup temporary files / stored packages / build artifacts, run ./cleanup_packages ./package-directories.txt from inside the scripts folder (cf. readme.md)

  2. Modify the projects/your_project/architect/build/options section in your angular.json: 

    {{ angular_json | async }}

  3. Add dependency to locally installed package node-module in package.json: 

    {{  package_json | async }}

  4. Add the necessary imports to your component (in case of standalone components) or app.module.ts

    In order to use Bindings, also import CommonModule and FormsModule 

    {{ component_imports | async }}

  5. build shared components by executing npm run build:shared-components from the workspace root.

    REMEMBER:every time you make a change to assets/code in the shared components library, it has to be rebuild before the changes are applied to the consuming projects.

    HINT: if changes are not applied after rebuild, try deleting the local Angular cache folder (.angular)
