Controls: Toggle

Simple toggle with configurable label and enabled/disabled state. Toggle data can be assigned using Two-Way DataBinding.

  • elementTitle sets the title
  • elementId sets id and name of the input toggle
  • use [(data)] property for displaying a toggle and bind to its value (Two-way-binding)
  • use (change) for callbacks when the bound dat value changes
  • [disabled] can be bound to set enabled state
  • BUG: Disabled + Active State is not correctly visualized
