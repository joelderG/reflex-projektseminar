Controls: Slider

Slider for range value input combined with text input and descriptive text on hover.

Short description of the values. Short description of the values of this slider. Minimum is bound to the value of the first slider.
  • elementTitle sets the label
  • elementId sets id and name of the input element
  • unit sets filed next to the text input
  • use [(data)] property for displaying a toggle and bind to its value (Two-way-binding)
  • use (change) for callbacks when the bound dat value changes
  • [min], [max] and [step] can be bound to specify value range and value change
  • content inside the element is displayed as description next to the slider
