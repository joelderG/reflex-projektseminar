Dropdown to choose between different options including label and short descriptive text.

Select one of the available Filters from the list.
Moving Average: simple average over the last n frames SavitzkyGolay: polynomial regression filter

  • use with class settings_item
  • elementTitle sets the label
  • elementId sets id and name of the input element
  • use [(data)] property for displaying a toggle and bind to its value (Two-way-binding)
  • use (change) for callbacks when the bound dat value changes
  • the description next to the slider is wrapped in a separate paragraph element with description attribute
